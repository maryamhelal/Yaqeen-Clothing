# Yaqeen-Clothing

## Install dependencies

### Backend

```bash
cd backend
```

```bash
npm install
```

#### Example .env

```bash
PORT = 5000
MONGO_URI=your_mongo_link
JWT_SECRET=secret-jwt-token
EMAIL_USER=your_email
EMAIL_PASS=your_app_password
```

### Frontend

```bash
cd frontend
```

```bash
npm install
```

## Running the app

### Backend

```bash
npm start
```

### Frontend

```bash
npm start
```

### Using Docker

```bash
docker compose up -d
```
